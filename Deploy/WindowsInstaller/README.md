# Создание установщика для Windows

Для создания скрипта установщика был использован [Inno Setup Compiler](https://jrsoftware.org/isdl.php).

## Публикация

1. Изменить версию в константе `MyAppVersion`.

1. Собрать установщик, нажав кнопку **Build -> Compile**.
   Установщик будет создан в директории скрипта.

Если в текущей версии будет изменена версия .NET, необходимо также изменить константу `PublishDirectory`.
